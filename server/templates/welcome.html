AutoCode is an innovative automatic coding tool designed to bootstrap any software project incrementally, transforming README.md instructions into a fully functional software project using Claude 3.5 Sonnet API.

Getting Started

  1. Log in to your account at https://autocode.work
  2. Create a new project or import an existing one
  3. Update your README.md with project instructions
  4. Use AutoCode to generate and modify your code

Key Features

  • Automatic code generation based on README.md instructions
  • Incremental project building
  • Self-updating README.md with new design ideas
  • Code quality checks and suggestions
